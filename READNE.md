# FuelMate

FuelMate is an java-based platform designed to help users monitor their Total Daily
Energy Expenditure (TDEE), track calorie intake and log exercise activity to support
their personal health and fitness goals. 

## Features
- Add food entries
- View food entries
- Update entries
- Delete entries
- GUI interface using Java Swing
- MySQL database integration
- INNER JOIN queries to combine user and food data

## Technologies Used
- Java
- JDBC
- MySQL
- Swing (GUI)

## How to Run
1. Set up MySQL database (FuelMate)
2. Update database credentials in DatabaseConnection.java
3. Run FuelMateGUI.java
